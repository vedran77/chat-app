<h1 align="center">ChatApp</h1>

> Simple Web Application for chatting with friends, **application is still in development**.

### ✨ [Demo] - Soon

## Install - client side
```sh
cd frontend
npm install
```

## Install - server side
```sh
npm install
```

## Usage - client side
```sh
cd frontend
npm run serve
```
## Usage - server side
```sh
npm run build:dev
```
